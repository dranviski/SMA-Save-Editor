ℹComo funciona o PNG
O jogo salva personagens como imagens .png. Além da foto visual da personagem, o arquivo contém dados embutidos com toda a aparência, traços de personalidade e atributos da personagem.

Para importar: use o botão Importar Talentos no jogo após adicionar via este editor, ou use diretamente o modo de "Entrevista Única" com o PNG exportado.
